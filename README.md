# ⚡ Vidyut AI — BESCOM Demand Forecasting & Theft Detection

Vidyut is an AI-powered utility intelligence platform designed for BESCOM (Bangalore Electricity Supply Company). It provides real-time demand forecasting using Ensemble models (Prophet + LightGBM + LSTM) and advanced theft detection using anomaly detection algorithms.

## 🚀 Deployment Features
- **API:** FastAPI powered backend
- **Dashboard:** Streamlit powered UI
- **Database:** Serverless Postgres (Neon)
- **Cache:** Serverless Redis (Upstash)
- **Compute:** Hugging Face Spaces (Docker)

## 🛠️ Tech Stack
- **Forecasting:** Facebook Prophet, LightGBM, PyTorch (LSTM)
- **Anomalies:** Isolation Forest, XGBoost
- **API:** FastAPI, Uvicorn
- **Dashboard:** Streamlit
- **Infrastructure:** Docker, Supervisord
